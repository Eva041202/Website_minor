Website: CSS to the Rescue

De opdracht CSS to the Rescue werd gegeven door Sanne en Nils.
Voor CSS konden we uit vier verschillende opdrachten kiezen. Ik had gekozen voor de Silly Walk.

Wat CSS lastiger maakte was dat we zonder classes, id's en JavaScript moesten werken.
Dacht ik net een beetje basis geleerd te hebben, kreeg ik dit voor m'n kiezen.

Ik wist eerst niet waar ik moest beginnen. Om het mezelf makkelijker te maken was het handig om een poppetje te maken uit verschillende onderdelen.
Maar ja hoe maak je dan die onderdelen? Ik had geen idee.
Joost heeft mij laten zien dat je dit met divs kan doen. In 4 weken tijd is het mij toch gelukt om iets animerend te maken.

De opdracht vond ik erg lastig en liet me vaak denken "waar ben ik aan begonnen". Maar dat houd me niet zomaar tegen.
Doordat deze opdracht zo'n uitdaging was heb ik erg veel geleerd, dat ik ook in andere opdrachten heb kunnen toepassen.
Zo heb ik o.a. :has weer toegepast in BT.
Ook heeft deze opdracht me laten zien dat ik zelf nooit wat met animatie ga doen. Dit ligt me toch wat minder.
