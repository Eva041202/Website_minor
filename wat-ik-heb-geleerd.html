Reflectie

Toen ik begon aan deze minor had ik vrijwel geen kennis met coderen.
Ik studeer aan de Pabo en mijn doel is om kleuterjuf te worden. Totaal iets anders.

Ik koos voor deze minor, omdat code mij al eerder interresant leek en ik mezelf wilde uitdagen om wat nieuws te leren.
In het begin vond ik dit best spannend, ik had geen codeer achtergrond zoals de rest, maar ik was ook erg enthousisast.
Al snel kreef ik ook het gevoel dat mijn leren te traag ging.
Soms denk ik dit nog steeds, maar als ik dan kijk waar ik vandaan kwam ben ik toch erg gegroeid.

Tijdens Sprint 0 was dit ook te zien.
De opdracht was om een website te bouwen waarmee je liet zien wat je al kon.
Mijn website was heel erg simpel. Toch ben ik ergens trots als ik dan die website vergelijk met wat ik nu kan.
Wie weet wat ik tijden en/of na de meesterproef kan.

Vanaf het begin wilde ik alles dat ik leerde bijhouden in een schrift.
De eerste paar weken was ik meer aan het schrijven in dat schrift dan aan het coderen.
Dat schrift gebruik ik nog steeds regelmatig. Zowel om nieuwe informatie te noteren als code terug vinden om weer te kunnen gebruiken.

Tijdens de opdrachten werd mijn HTML en CSS steeds beter. Natuurlijk is er nog veel te leren, maar ik heb nu enigszins een goede basis.
Vooral met CSS ben ik enorm gegroeid. Ik begon met alleen een background-color: kunnen maken en nu begrijp ik styling meer, wat toegankelijkheid en responsiveness.

Wat ik tijdens de eerste opdrachten heb geleerd kon ik ook weer toepassen tijdens de Hackaton.
Naast dat HTML en CSS steeds beter gaan blijf ik moeite hebben met JavaScript.
JavaScript kan ik nog niet zelfstandig. Ik moet hiervoor altijd docenten en/of chatgpt voor uitleg vragen.
De uitleg is te begrijpen, maar het zelfstandig toepassen blijft lastig. Hier wil ik dan ook meer mee oefenen.

Ik heb erg veel zin om te beginnen aan de meesterproef, maar hier ben ik ook weer erg nerveus voor.
Het is dan wel een geruststelling dat we dit in groepsverband doen. Met een duidelijke taakverdeling denk ik wel dat het me gaat lukken.

Hoewel ik hier later niet in ga werken, omdat ik de pabo doe, ben ik wel erg blij met mijn keuze voor een minor.
Ik wil mezelf altijd uitdagen door nieuwe dingen te leren en een beetje uit de "comfort bubble" komen.
De minor heeft me veel nieuwe inzichten gegeven. Ik kan nu anders naar een website kijken en ik zie in wat een werk het allemaal wel niet is.

Doelen meesterproef

Meer durven vragen bij medestudenten.
Dit wil ik doen door vragen die ik heb direct te stellen. Niet wachten en afvragen of het misschien een domme vraag is.
Als ik het niet vraag kom ik het nooit te weten.

Binnen het project oefenen met (eenvoudige) JavaScript.
Dit kan ik doen door online oefeningen met JavaScript te proberen.
Ook als iets tijdens de meesterproef met JavaScript gedaan moet worden kan ik de uitdaging aangaan door dit op mij te nemen, inplaats die bij iemand anders neer te leggen.

Wat meer experimenteren met CSS.
HTML en CSS zijn nog steeds in ontwikkeling, zodat er minder JavaScript gebruikt hoeft te worden.
Nieuwe code dat uitkomt lees ik wel eens, maar pas ik nooit toe.
Ik wil iets van de nieuw uitgekomen code toepassen.
