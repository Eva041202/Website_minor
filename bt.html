Website: Browser Tech

De opdracht Browser Tech werd gegeven door Vasilis en Victor.
Voor BT hebben we een erfbelastingsformulier deels na moeten maken met de huisstijl van NS.

Tijdens deze opdracht was het belangrijk om na te denken over eventuele valkuilen in een formulier.
Het erfbelastingsformulier op de website van de overheid is gemaakt voor op papier. Wij konden natuurlijk met code dit gebruiksvriendelijker en overzichtelijker maken.
Door onder andere vragen te laten verschijnen en verdwijnen wanneer nodig is, werd er minder ruimte gebruikt en bleef het overzichtelijk.

Met deze opdracht heb ik veel nieuwe code geleerd.
Zo heb ik geleerd hoe je een formulier maakt, hoe je in css met :has werkt (voor de verschijnende vragen), verschillende letter typen (dikte, schuin) en werken met <details>
