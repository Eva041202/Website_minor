Over de spreker:

Kilian Valkhof is een Nederlandse developer en designer.
Hij is vooral bekend met zijn browser Polypane.

Lees meer...

Polypane is een browser voor webdevelopers.
Je kan één website tegelijk zien in meerdere schermgroottes.

Kilian richt zich op responsive design, accessibility, performance en developer tooling.

Al jong richtte hij een webbureau op, dat groeide tot zo'n 14 medewerkers.
Na jaren websites bouwen begon hij zich te ergeren aan browsers waarbij je je viewportgroottes moest wisselen tijdens dat je aan het bouwen was.
Op die manier is uiteindelijk Polypane ontstaan.

Naast Polypane heeft Kilian aan meer gewerkt:

Kilian heeft ook een persoonlijke website (kilianvalkhof.com) waar hij regelmatig nieuwe bevindingen schrijft.

Wat neem ik mee:

Dit was de eerste Weekly Nerd van de minor.
Ik was net begonnen aan de minor en had bijna geen kennis.

Lees meer...

Daardoor ging een groot deel van de lezing eerlijk gezegd nog boven mijn hoofd.
Met behulp van m'n aantekeningen is een deel wel bijgebleven.

Kilian vertelde o.a. over hoe developers gaten en oplossingen zoeken in HTML en CSS om minder JavaScript te gebruiken.
Veel JavaScript kost veel energie voor een browser om dat allemmaal te laden.
Minder JavaScript zorgt voor een stabielere en snellere website.

Tijdens de minor leerde ik ook, vooral met CSS, wat mogelijk was met code in HTML en CSS om zo min mogelijk JavaScript te gebruiken.
Hier maakte ik dan ook goed gebruik van, aangezien ik JavaScript een beetje ontweek.
Alleen als het écht moest ging ik wat met JavaScript doen.

Tijdens de lezing liet Kilian ook zien hoe je met HTML en CSS een switch kan maken. Bijv. een slider voor dark mode.
Dit gaf mij gelijk het idee om op mijn website voor Sprint 0 ook een slider te maken.
Kilian ging voor mij tijdens de lezing veels te snel en heb daardoor niet alle code hiervoor kunnen oppikken.
Ook wat er online te vinden was begreep ik niet allemaal en de slider werd voor mij een groot drama, waardoor ik dat niet op mijn website heb.

Dit was even een reality check voor mezelf.
Ik dacht dat doe ik wel eventjes.
